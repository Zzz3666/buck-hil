## 1.2 时钟域规划

| 时钟域 | 频率 | 源 | 用途 |
|--------|------|-----|------|
| `clk_pl_400` | 400 MHz | MMCM (PS→PL clock) | PWM 捕获定时器 |
| `clk_pl_100` | 100 MHz | MMCM | Buck 求解器 + DAC 接口 |
| `clk_axi` | 100 MHz | PS AXI interconnect | AXI-Lite 寄存器接口 |
| `clk_dac_spi` | 50 MHz | MMCM (clk_pl_100 分频) | DAC SPI 接口 |

**跨时钟域处理**：

| 跨域路径 | 方向 | 处理方式 |
|----------|------|----------|
| `clk_axi` → `clk_pl_100` | PS→PL (参数写入) | 双缓冲寄存器 + 握手机制 |
| `clk_pl_100` → `clk_axi` | PL→PS (数据 DMA) | AXI-Stream → DMA FIFO |
| `clk_pl_400` → `clk_pl_100` | PWM捕获→求解器 | 同步链 (2级FF) + 脉冲展宽 |
| `clk_pl_100` → `clk_dac_spi` | 求解器→DAC | 同步 FIFO (深度 16) |

## 1.4 资源估算 (ZU3EG)

ZU3EG 资源总量 vs 本设计占用：

| 资源 | ZU3EG 总量 | 本设计估计 | 利用率 |
|------|-----------|-----------|--------|
| System Logic Cells | 154,350 | ~3,000 | < 2% |
| CLB LUTs | 70,560 | ~1,200 | < 2% |
| CLB Registers | 141,120 | ~2,000 | < 2% |
| Block RAM (36Kb) | 216 | ~8 | 4% |
| DSP48 | 360 | ~6 | < 2% |

**结论**：ZU3EG 远超出当前需求。剩余资源足够支持：
- 多相 Buck 扩展（4~8 相）
- LLC 谐振变换器（状态变量翻倍）
- 更高开关频率 → 更细仿真步长（10ns）
- 硬件故障注入逻辑

## 1.5 延迟分析

| 链路环节 | 延迟 | 说明 |
|----------|------|------|
| PWM 边沿 → 占空比有效 | ~10 ns | 2 级同步 + 边沿检测 |
| 占空比 → 求解器输出更新 | 100 ns | 一个求解步长 |
| 求解器 → DAC 输出 | ~500 ns | SPI 传输 24bit @ 50MHz + DAC 建立 |
| DAC 输出 → 模拟前端 | ~100 ns | 运放传播延迟 (SGM8061: 10ns typ) |
| 模拟前端 → 控制器 ADC | 取决于控制器 | 控制器采样频率 |
| **总闭环延迟** | **< 1 μs** | 远小于 200kHz 开关周期 (5μs) |

## 1.7 关键风险矩阵

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| PWM 捕获毛刺导致错误占空比 | 中 | 高 | 5ns 数字滤波 + 3周期一致性检查 |
| DAC 台阶引入高频杂散 | 高 | 中 | 2阶 RC 重建滤波 (fc=500kHz) |
| 求解器数值发散 (i_L < 0) | 低 | 高 | 下界钳位 + 饱和处理 |
| TCP 粘包/半包导致帧错位 | 高 | 中 | 状态机逐字节解析，不依赖 Read 边界 |
| 模拟前端地回路 | 中 | 中 | 单点接地，AGND/DGND 经 0Ω 连接 |
| ZU3EG 电源纹波耦合到 DAC | 中 | 中 | 独立 LDO 给 DAC 供电，测量纹波 < 5mVpp |
| 时序不收敛 (跨域路径) | 低 | 低 | 严格用同步链/异步 FIFO，STA 验证 |
