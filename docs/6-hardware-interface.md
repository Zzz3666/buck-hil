## 6.3 模拟前端设计

### 6.3.1 Vout 通道 (0~12V 输出)

DAC80508 输出 0~5V (内部基准 2.5V, gain=2 模式)，经运放放大到 0~12V。

```
DAC80508 VOUT0 (0~5V, 内部缓冲, ±15mA 驱动)
        │
        ▼
        ┌──────────┐
        │ SGM8061  │  同相放大, G = 2.4
        │ +IN ─────┼── 来自 DAC VOUT0
        │ -IN ──┬──┼── R2 = 10k → GND
        │       │  │
        │ OUT ──┴──┼── Rf = 14k → -IN
        └──────┬───┘
               │  0~12V
               ▼
        2阶 RC 低通 (fc = 500 kHz):
        ┌─────┐    ┌─────┐
        │ R3  │────│ R4  │──── BNC VOUT
        │ 330Ω│    │ 330Ω│
        └──┬──┘    └──┬──┘
           │ C1       │ C2
          ─┴─ 1nF    ─┴─ 1nF
           ─┬─        ─┬─
            │ GND      │ GND

增益: G = 1 + Rf/R2 = 1 + 14k/10k = 2.4
验证: 5V × 2.4 = 12.0V ✓
LPF fc = 1 / (2π × 330 × 1nF) ≈ 482 kHz ≈ 500 kHz
```

DAC80508 自带输出缓冲，DAC 到运放之间不需要额外缓冲——直接连接即可。

### 6.3.2 I_L 通道 (0~10A → 0~5V 等效)

I_L 使用相同的 0~5V 输出范围，增益 = 1.0 (跟随器)。

```
DAC80508 VOUT1 (0~5V) ——表示→ 0~10A
  → 运放跟随器 (G=1, 阻抗匹配)
  → 2阶 RC LPF (fc = 500 kHz)
  → BNC I_L (0~5V, 表示 0~10A 电流)
```

与 Vout 通道的区别：DAC 的 5V 满量程直接映射到 10A，无需额外放大——控制器端如果期望不同电压量程，由其自身 ADC 前端处理。

### 6.3.3 PWM 输入调理

```
DUT PWM 输出 (3.3V 或 5V CMOS)
        │
        ▼
    BNC 输入 (50Ω 端接可选)
        │
        ▼
   ┌─────────┐
   │ R1 = 1k │───── 限流
   └────┬────┘
        │
        ├─────────────────────────► ZU3EG PL I/O
        │
   ┌────┴────┐
   │TVS 3.3V │──── 钳位到 3.3V (SMBJ3.3A 或类似)
   └────┬────┘
        │
       GND
```

如果 DUT 输出为 5V：
- 分压: R1=2k, R2=3k → Vout = 5V × 3/(2+3) = 3.0V (安全)

## 6.4 电源树

```
12V DC 输入
  │
  ├──► 5V DCDC (降压) ──► ZU3EG 载板 5V 输入
  │     │                   │
  │     │                   ├──► ZU3EG 板载 PMIC → VCCINT(0.85V), VCCAUX(1.8V), VCCO(3.3V), etc.
  │     │                   │
  │     │                   ├──► DVDD_DAC: 3.3V (与 PL Bank VCCO 同源)
  │     │                   │
  │     │                   └──► 独立 LDO (LP5907-5.0) → 5V → DAC80508 AVDD (低噪声)
  │     │
  │     └──► ±12V DCDC (升压) ──► ±5V LDO (ADP7118) ──► SGM8061 ±VDD

说明:
  DAC80508 仅需单电源 (2.7~5.5V)，不需要 AD5686 那样的 ±15V 高压轨。
  AVDD 使用独立 LDO 从 5V 取电→5V 输出 (LP5907 有 5.0V 输出版本)，
  或者直接用 TPS7A4700 (可调) 出 5V 低噪轨。
  SGM8061 运放因需输出 12V，仍需要 ±12V 供电 → ±5V 即可驱动 12V 满摆幅。

对比旧方案 (AD5686):
  旧: 12V → 5V → LP5907-3.3 → AD5686 AVDD + SGM8061 ±5V
  新: 12V → 5V → LP5907-5.0 → DAC80508 AVDD + SGM8061 ±5V
  省掉了: 无需 ±15V (DAC81408 才需要)
```

## 6.7 布局注意事项

1. **DAC 模拟路径**：DAC80508 VOUT → 运放 → BNC 路径短 (< 25mm)，远离数字总线
2. **地平面**：整板 4 层，第 2 层完整 GND 平面。AGND 和 DGND 在 DAC80508 下方经 0Ω 单点连接
3. **SPI 信号**：SCLK 串联 22Ω 终端电阻以减少振铃。DAC80508 为 32-bit SPI 帧，比 AD5686 长 33%
4. **PWM 输入**：BNC → TVS → 分压 → PL pin，路径 < 30mm，远离 DAC 模拟部分
5. **去耦**：DAC80508 AVDD 靠近引脚放置 100nF + 1μF NP0 + 10μF 钽；REFIO 外接 100nF 旁路
6. **LDAC**：若不需要同步更新多通道，可将 LDAC 接 GND（自动更新模式）
7. **散热**：WQFN-16 底部焊盘必须接地，手工焊接需热风枪/加热台

## 6.8 验证清单

搭建完成后，逐一验证：

- [ ] 电源轨电压正确 (5V AVDD, 3.3V DVDD, ±5V 运放; 纹波 < 20mVpp)
- [ ] DAC80508 SPI 通信正常 (写 0x8000 → VOUT=2.5V; 写 0xFFFF → VOUT=5.0V)
- [ ] 运放增益正确 (输入 2.5V → 输出 6.00V; 输入 5.0V → 输出 12.00V)
- [ ] PWM 输入 200kHz/50% → PL 捕获占空比 = 0.500
- [ ] Buck 求解器开环运行 (固定 duty=50%, 参数默认值, 输出收敛)
- [ ] 闭环测试: 信号发生器产生 PWM → DAC 输出 Vout → 示波器观察
- [ ] PS↔PC TCP 通信: ping → 参数读写 → 数据流
- [ ] 24 小时烤机: 无帧丢失, 无 CRC 错误, 无内存泄漏
