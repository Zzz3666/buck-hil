## 6.3 模拟前端设计

### 6.3.1 Vout 通道 (0~12V 输出)

```
AD5686 VOUTA (0~2.5V)
        │
        ▼
   ┌─────────┐
   │ R1 = 10k│─────┐
   │         │     │
   └─────────┘     │
                   │    ┌──────────┐
                   ├────┤ SGM8061  │
                   │    │ +IN      ├──────┐
   ┌─────────┐     │    │ -IN ─────┤ Rf   │
   │ R2 = 2.63k│───┘    │ OUT ─────┼──┐   │
   │  (GND)  │          └──────────┘  │   │
   └─────────┘                        │   │
                                      │   │
                        ┌─────────────┘   │
                        │                 │
                        │    ┌─────┐      │
                        └────┤ Rf  │──────┘
                             │38.3k │
                             └─────┘

增益: G = 1 + Rf/R2 ≈ 1 + 38.3/2.63 ≈ 15.56

等等 — 目标: 0~2.5V → 0~12V, G = 12/2.5 = 4.8

修正:
  R2 = 10k (到地)
  Rf = 38k  →  G = 1 + 38/10 = 4.8 ✓

运放输出 → 2阶 RC 低通滤波 (fc = 500 kHz):
  ┌─────┐    ┌─────┐
  │ R3  │────│ R4  │──── BNC VOUT
  │ 330Ω│    │ 330Ω│
  └──┬──┘    └──┬──┘
     │ C1       │ C2
    ─┴─ 1nF    ─┴─ 1nF
    ─┬─        ─┬─
     │ GND      │ GND

  fc = 1 / (2π × 330 × 1e-9) ≈ 482 kHz ≈ 500 kHz
```

### 6.3.2 I_L 通道 (0~10A → 0~2.5V 等效)

与 Vout 通道相同的运放结构，仅量程不同。

```
AD5686 VOUTB (0~2.5V) ——表示→ 0~10A (增益通过前端设置)
```

实际上，I_L 的量程由 DAC 的满量程和运放增益共同决定。这里假设电流检测电阻为 0.1Ω，10A → 1V → 需要增益 1.0。

```
增益 = 1.0 → Rf = 0 (跟随器)

运放输出 → BNC I_L (0~2.5V, 表示 0~10A 电流信号)
```

### 6.3.3 PWM 输入调理

```
DUT PWM 输出 (3.3V 或 5V CMOS)
        │
        ▼
    BNC 输入 (50Ω 端接可选)
        │
        ▼
   ┌─────────┐
   │ R1 = 1k │───── 限流
   └────┬────┘
        │
        ├─────────────────────────► ZU3EG PL I/O
        │
   ┌────┴────┐
   │TVS 3.3V │──── 钳位到 3.3V (SMBJ3.3A 或类似)
   └────┬────┘
        │
       GND
```

如果 DUT 输出为 5V：
- 分压: R1=2k, R2=3k → Vout = 5V × 3/(2+3) = 3.0V (安全)

## 6.4 电源树

```
12V DC 输入
  │
  ├──► 5V DCDC (降压) ──► ZU3EG 载板 5V 输入
  │     │                   │
  │     │                   ├──► ZU3EG 板载 PMIC → VCCINT(0.85V), VCCAUX(1.8V), VCCO(3.3V), etc.
  │     │                   │
  │     │                   └──► 3.3V LDO (LP5907) ──► AD5686 AVDD/DVDD (独立, 低噪声)
  │     │
  │     └──► ±12V DCDC (升压) ──► ±5V LDO (ADP7118) ──► SGM8061 ±VDD
  │
  └──► 12V 直接 ──► 运放 VDD (如果运放支持 12V 单电源, 如 SGM8061 上限 5.5V 则不可)

实际方案:
  12V → 5V DCDC → TPS7A4700 (超低噪声 LDO) → 3.3V → AD5686
  12V → 5V DCDC → ADM660 (电荷泵) → ±5V → SGM8061
```

## 6.7 布局注意事项

1. **DAC 模拟路径**：AD5686 VOUT → 运放 → BNC 路径短 (< 25mm)，远离数字总线
2. **地平面**：整板 4 层，第 2 层完整 GND 平面。AGND 和 DGND 在 AD5686 下方经 0Ω 单点连接
3. **SPI 信号**：SCLK 串联 22Ω 终端电阻以减少振铃
4. **PWM 输入**：BNC → TVS → 分压 → PL pin，路径 < 30mm，远离 DAC 模拟部分
5. **去耦**：每个 IC 的 VDD 靠近引脚放置 100nF + 10μF (AD5686 用 100nF + 1μF NP0)
6. **ZU3EG 连接**：高速信号 (SPI) 用 FMC 或高速排针，长度匹配 < 5mm

## 6.8 验证清单

搭建完成后，逐一验证：

- [ ] 电源轨电压正确 (3.3V, 5V, ±5V 纹波 < 20mVpp)
- [ ] AD5686 SPI 通信正常 (写 0x8000 → VOUT=1.25V)
- [ ] 运放增益正确 (输入 1.25V → 输出 6.00V)
- [ ] PWM 输入 200kHz/50% → PL 捕获占空比 = 0.500
- [ ] Buck 求解器开环运行 (固定 duty=50%, 参数默认值, 输出收敛)
- [ ] 闭环测试: 信号发生器产生 PWM → DAC 输出 Vout → 示波器观察
- [ ] PS↔PC TCP 通信: ping → 参数读写 → 数据流
- [ ] 24 小时烤机: 无帧丢失, 无 CRC 错误, 无内存泄漏
